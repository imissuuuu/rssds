# tinyxml2 を配置する

このディレクトリに以下の2ファイルを配置してください。

## ダウンロード方法

```bash
cd lib/tinyxml2
curl -LO https://raw.githubusercontent.com/leethomason/tinyxml2/master/tinyxml2.h
curl -LO https://raw.githubusercontent.com/leethomason/tinyxml2/master/tinyxml2.cpp
```

または GitHub からZIPをダウンロードして `tinyxml2.h` と `tinyxml2.cpp` だけを配置。
