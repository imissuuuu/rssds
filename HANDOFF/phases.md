## Ideas

| Idea | メモ |
|------|------|
| リッチテキスト表示 | 見出し・本文・絵文字など Markdown 相当の表現（Phase 0 基盤の上に構築） |
| フォント対応 | フォント選択・サイズ変更 |
| 設定機能 | テーマ（ダーク/ライト）、表示設定、その他もろもろ |
| 画像表示 | 記事内画像のレンダリング |
| ログ機能 | 本文抽出失敗ページの記録・改善補助 |
| 既読/未読管理 | 記事の既読状態を保持 |
| スター/ブックマーク | 記事をお気に入り登録 |
| カテゴリ/フォルダ | フィードをグループ分け |
| オフライン読書 | 記事をキャッシュしてオフライン閲覧 |
| OPML対応 | フィードリストの標準形式でインポート/エクスポート |
| キーワードフィルタ | 特定トピックを除外/優先表示 |
| 自動更新設定 | フィード更新間隔の設定 |
| フィードURL管理 | QRコード読み取りによるURL入力等 |
