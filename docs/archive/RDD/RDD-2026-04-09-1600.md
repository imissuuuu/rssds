# RDD - 2026-04-09-1600
**Project Name:** 3DS RSS Reader (MVP)

## 1. 機能一覧

### 機能要件
- FR-1: SDカード上の `feeds.txt` からフィードURL読み込み
- FR-2: HTTPS経由でRSSフィード取得（libcurl + mbedtls）
- FR-3: XMLパースして記事タイトル一覧表示（tinyxml2）
- FR-4: 記事選択 → 本文表示（HTML簡易タグストリップ）

### 非機能要件
- NFR-1: 3DS実機で動作
- NFR-2: 日本語表示対応（システムフォント使用）

## 2. 詳細仕様

MVP段階のため最小限：
- `sdmc:/3ds/rssreader/feeds.txt` から1行1URLで読み込み
- 各URLに対しHTTP GETでRSS/Atom XMLを取得
- `<item>` or `<entry>` をパースしてタイトル・本文を抽出
- 本文はHTMLタグを除去してプレーンテキスト化
- RSS 2.0 / Atom 両対応

## 3. UI/UX考慮事項

MVP段階では最小限のUI。設計フェーズで決定。

## 4. 追加設計情報

- 通信: libcurl (devkitPro同梱) + mbedtls でTLS対応
- XML: tinyxml2（ソース同梱）
- HTMLタグ除去: 簡易ストリッパー（自前）、不足なら後でOSSに差し替え
- 目的: 実機で動かして制約を特定した後、本格的な要件定義を行う
