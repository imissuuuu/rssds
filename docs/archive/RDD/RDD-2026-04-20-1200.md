# RDD - 2026-04-20-1200
**Project Name:** Phase 7.7 本文中インライン画像

## 1. 機能一覧

### 機能要件
- **FR-1**: 記事本文中の `<img>` タグをテキスト行内にプレースホルダーとして挿入・表示
- **FR-2**: 記事オープン時に `<img>` の画像を低解像度で自動ダウンロード・インライン表示
- **FR-3**: ダウンロード中は既存のプログレスバーUIをインライン表示
- **FR-4**: インライン表示済み画像にフォーカスしてAボタン → 既存 ImageView へ遷移（高解像度ロード・ズーム・パン）
- **FR-5**: 1記事内の複数 `<img>` タグを全て対象とする（枚数は制約次第で上限設定）

### 非機能要件
- **NFR-1**: 自動ロード中もテキストスクロールを阻害しない（非同期 or バックグラウンドロード）
- **NFR-2**: 低解像度ロードはRAM使用量を考慮したサイズ上限を設ける（上限: 幅320px相当）

## 2. 詳細仕様

### FR-1: プレースホルダー挿入
- 入力: HTML本文パース時に検出した `<img src="...">` タグ
- 処理: テキスト行リストに `ImagePlaceholder` 型の行を挿入（src URL を保持）
- 出力: ArticleView のスクロールラインに `[ 画像 ]` 的な表示行として現れる

### FR-2: 自動インラインロード
- 入力: 記事オープンイベント
- 処理: 各プレースホルダーの URL に対して低解像度でHTTPS取得 → テクスチャ生成
- 出力: プレースホルダーが実画像に置き換わる
- サイズ: 元の width/height 属性を尊重しつつ、幅320px・高さ上限を超える場合はアスペクト比維持でリサイズ

### FR-3: ローディングインジケーター
- 入力: ダウンロード開始〜完了
- 処理: 既存プログレスバーコンポーネントを流用してインライン表示
- 出力: ダウンロード完了後にプログレスバーが消えて画像が表示される

### FR-4: ImageView遷移
- 入力: インライン表示済み画像行にフォーカスし Aボタン押下
- 処理: 既存 `Screen::ImageView` に URL を渡して遷移
- 出力: 高解像度（1024px）でズーム・パン可能なImageViewが開く

## 3. UI/UX考慮事項

### ArticleView スクロール（概念図）
```
┌─────────────────────────────┐
│ 記事本文テキスト...          │
│ テキスト続き...              │
│ ┌─────────────────────────┐ │
│ │  [====　プログレス　  ] │ │  ← ロード中
│ └─────────────────────────┘ │
│ テキスト続き...              │
│ ┌─────────────────────────┐ │
│ │  [画像インライン表示]    │ │  ← ロード済（Aボタンで ImageView）
│ └─────────────────────────┘ │
└─────────────────────────────┘
```

### ボタン操作
| 状態 | ボタン | 動作 |
|------|--------|------|
| 画像行にフォーカス（ロード済） | A | ImageView へ遷移 |
| 画像行にフォーカス（ロード中） | A | 無効（ロード完了まで待つ） |

## 4. 追加設計情報

### 実装上の制約
- 3DS の RAM 制限（最大数MBのヘープ）により、同時展開テクスチャ枚数に上限を設ける
- 現在の ImageView は 1024px 高解像度テクスチャを保持するため、インライン用は別テクスチャ（低解像度）を管理
- libcurl による HTTPS 取得は既存の `fetchImageData()` 系関数を流用

### 既存コードへの影響（概要のみ）
- `ArticleView`: スクロール行リストに画像行タイプを追加、フォーカス処理・Aボタン処理を拡張
- `html_parser` / `content_extractor`: `<img>` タグのsrc抽出を追加
- `ImageDownloader` 系: インライン用低解像度バリアントを追加、または引数で制御
- `ProgressBar` UI: 既存コンポーネントをインライン位置に描画できるよう流用
