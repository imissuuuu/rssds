## 1. 全体方針

- **シングルスレッド**: MVP段階ではメインループ1本。通信中はブロッキング（ローディング表示）
- **citro2dでUI描画**: テキスト描画はシステムフォント、`C2D_TextParse` 毎フレーム呼び出し
- **tinyxml2同梱**: `lib/tinyxml2/` にソースを配置、Makefileでビルド対象に含める
- **状態遷移**: enum で画面状態を管理（フィード一覧 → 記事一覧 → 記事本文）

## 4. 各機能の詳細設計

### FR-1: feeds.txt 読み込み

```
起動時:
  fopen("sdmc:/3ds/rssreader/feeds.txt")
  → 1行ずつ読み、空行・#始まりをスキップ
  → vector<string> に格納
  ファイルが無い場合 → statusMsg にエラー表示して続行
```

### FR-2: HTTPS取得

```
httpGet(url):
  curl_easy_init()
  CURLOPT_URL           = url
  CURLOPT_SSL_VERIFYPEER = 1
  CURLOPT_CAINFO        = "sdmc:/3ds/rssreader/cacert.pem"
  CURLOPT_FOLLOWLOCATION = 1
  CURLOPT_WRITEFUNCTION = writeCallback  // std::string に追記
  CURLOPT_TIMEOUT       = 15
  curl_easy_perform()
  curl_easy_cleanup()
```

### FR-3: XMLパース

```
parseFeed(xml):
  doc.Parse(xml)
  root = doc.RootElement()

  if root->Name() == "rss":          // RSS 2.0
    channel = root / "channel"
    feed.title = channel / "title"
    for each "item":
      title   = item / "title"
      link    = item / "link"
      content = item / "content:encoded" ?? "description"
      pubDate = item / "pubDate"

  elif root->Name() == "feed":       // Atom
    feed.title = root / "title"
    for each "entry":
      title   = entry / "title"
      link    = entry / "link" @href
      content = entry / "content" ?? "summary"
      pubDate = entry / "published" ?? "updated"
```

### FR-4: HTML→テキスト変換

```
stripHtml(html):
  ステートマシン: OUTSIDE_TAG / INSIDE_TAG
  <p> <div> <br> <br/> <li> <h1-h6> → 改行挿入
  HTMLエンティティ → デコード
  連続改行・空白を正規化
```

### UI 画面レイアウト

```
上画面 (400x240)          下画面 (320x240)
┌────────────────────┐   ┌──────────────────┐
│ [FeedList]         │   │ > Feed A         │
│ フィード名表示      │   │   Feed B         │
│                    │   │   Feed C         │
│ [ArticleList]      │   │                  │
│ 記事タイトル表示    │   │ Up/Down:移動     │

│ [ArticleView]      │   │ START:終了       │
│ 本文スクロール表示  │   └──────────────────┘
└────────────────────┘

操作:
  D-Pad 上下: カーソル移動 / スクロール
  A: 決定（次画面へ）  ※遷移後 kDown &= ~KEY_A でキー消費
  B: 戻る             ※遷移後 kDown &= ~KEY_B でキー消費
  START: 終了
```

## 5. 実装順序

| Step | 内容 | ビルド確認 |
|------|------|-----------|
| 1 | Makefile + main.cpp（空ループ、gfx初期化/終了のみ） | 実機起動→START終了 |
| 2 | net.cpp（SOC + libcurl HTTPS取得） | 下記URLをハードコードしてフェッチ→コンソール出力 |
| 3 | feed_config.cpp（feeds.txt読み込み） | SD上のURLを読み込み確認 |
| 4 | rss.cpp + tinyxml2（XMLパース） | タイトル一覧をコンソール出力 |
| 5 | html_strip.cpp（タグ除去） | 本文テキスト化をコンソール出力 |
| 6 | ui.cpp（citro2d描画、3画面遷移） | 完成形のUI動作確認 |

> Step 1〜5 は `printf` でコンソール確認。Step 6 で citro2d UI に切り替え。
