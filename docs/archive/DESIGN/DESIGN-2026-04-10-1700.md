## 1. 全体方針

- `AppState` を「設定（ネットワーク不要）」と「ロード済みデータ」に分離する
- `Screen::Loading` を追加し、1フレーム描画→次フレームでフェッチする同期ローディングパターンを採用
- テキストスタイルを列挙型で管理し、`CHARS_PER_LINE` を画面サイズから計算する
- `feed_config` を JSON 対応に差し替え。`nlohmann/json` を `JSON_NOEXCEPTION` + `allow_exceptions=false` で使用

enum class TextStyle { Body, Heading };

struct StyleParams {
    float scale;
    u32   color;
};

static StyleParams resolveStyle(TextStyle style) {
    switch (style) {
        case TextStyle::Heading:
            return { TEXT_SCALE * 1.3f, CLR_TITLE };
        case TextStyle::Body:
        default:
            return { TEXT_SCALE, CLR_TEXT };
    }
}

static void drawStyledText(const char* str, float x, float y,
                           float z, TextStyle style) {
    StyleParams p = resolveStyle(style);
    drawText(str, x, y, z, p.scale, p.scale, p.color);
}

### `source/feed_config.cpp`（全面置き換え）

```cpp
#define JSON_NOEXCEPTION
#include "feed_config.h"
#include "nlohmann/json.hpp"
#include <cstdio>

std::vector<FeedConfig> loadFeedConfig(const std::string& path) {
    std::vector<FeedConfig> configs;

    FILE* fp = fopen(path.c_str(), "r");
    if (!fp) return configs;

    fseek(fp, 0, SEEK_END);
    long size = ftell(fp);
    rewind(fp);
    if (size <= 0) { fclose(fp); return configs; }

    std::string content(size, '\0');
    fread(&content[0], 1, size, fp);
    fclose(fp);

    // allow_exceptions=false: パース失敗時は discarded value を返す
    auto j = nlohmann::json::parse(content, nullptr, false);
    if (j.is_discarded()) return configs;
    if (!j.contains("feeds") || !j["feeds"].is_array()) return configs;

    for (const auto& item : j["feeds"]) {
        if (!item.contains("url") || !item["url"].is_string()) continue;
        FeedConfig cfg;
        cfg.url              = item["url"].get<std::string>();
        cfg.name             = item.value("name", cfg.url);
        cfg.fetch_full_text  = item.value("fetch_full_text", false);
        configs.push_back(std::move(cfg));
    }
    return configs;
}
```

## 4. 各機能の詳細設計

### FR-1: オンデマンドロード フロー

```
aptMainLoop()
  │
  ├─ currentScreen == Loading?
  │     ├─ feedLoaded[idx] == false → httpGet → parseFeed
  │     │     ├─ 成功: feedLoaded[idx]=true, screen=ArticleList
  │     │     └─ 失敗: statusMsg=エラー, screen=FeedList
  │     └─ feedLoaded[idx] == true → screen=ArticleList
  │          （キャッシュ済みフィードを再表示）
  │
  ├─ hidScanInput / uiHandleInput
  │     └─ FeedList + KEY_A → statusMsg=フィード名, screen=Loading
  │
  └─ uiDraw / C3D_FrameEnd
```

### FR-2: CHARS_PER_LINE の計算根拠

| パラメータ | 値 | 根拠 |
|-----------|-----|------|
| `TOP_W` | 400px | 上画面幅 |
| `TEXT_MARGIN_X` | 6px | 左右マージン |
| `CHAR_WIDTH_AT_SCALE1` | 12.0f | システムフォント実測近似値 |
| `TEXT_SCALE` | 0.5f | 現行スケール |
| **結果** | **(400 - 12) / (12 * 0.5) = 64** | ≈ 現行ハードコード 65 と近い |

> `CHAR_WIDTH_AT_SCALE1` は実機確認後に調整可能。計算式を変えずに定数を1箇所直すだけでよい。
